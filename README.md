# Enterprise IAM Lab — Microsoft Entra ID

An end-to-end, hands-on enterprise Microsoft Entra ID lab that simulates real-world user lifecycle, privileged access, conditional access, governance, and automation.

---

## About This Lab

This lab simulates the identity and access management infrastructure of a fictional organization, Lane Technologies, built entirely in Microsoft Entra ID. The project reflects real-world IAM workflows and security controls used in enterprise environments and are designed to demonstrate both technical depth and security reasoning across the full identity lifecycle.

The lab was built by drawing on hands-on experience managing identity governance, access controls, and endpoint security in a regulated healthcare environment — translating real-world knowledge into a documented, reproducible enterprise IAM program.

---

## Technologies Used

- Microsoft Entra ID (Azure AD)
- Conditional Access
- Privileged Identity Management (PIM)
- Identity Governance & Access Reviews
- Role-Based Access Control (RBAC)
- Microsoft Azure Portal
